Division, Floor Div, and Modulo

以下是 Python 3 中与除法相关的运算符的示例：

真除法：/（小数除法）：

>>> 1 / 5
0.2

>>> 25 / 4
6.25

>>> 4 / 2
2.0

>>> 5 / 0
ZeroDivisionError

向下取整除法： //（整数除法）：

>>> 1 // 5 # truncate result of true division
0

>>> 25 // 4
6

>>> 4 // 2
2

>>> 5 // 0
ZeroDivisionError

（整数除法）模数：%（余数）：

>>> 1 % 5
1

>>> 25 % 4
1

>>> 4 % 2
0

>>> 5 % 0
ZeroDivisionError

除以 0 时会发生 ZeroDivisionError 。

涉及 % 运算符的一项有用技术是检查数字 x 是否可以被另一个数字 y 整除：

x % y == 0

例如，为了检查 x 是否为偶数：x % 2 == 0

Return and Print

您定义的大多数函数都包含一个 return 语句，该语句提供用于调用该函数的调用表达式的值。 当 Python 执行 return 语句时，函数调用会立即终止。如果 Python 到达函数体的末尾而没有执行 return 语句，则函数返回 None 。

相反， print 函数用于显示值。与 return 语句不同，当 Python 评估对 print 的调用时，该函数不会立即终止。

def what_prints():
    print('Hello World!')
    return 'Exiting this function.'
    print('61A is awesome!')

>>> what_prints()
Hello World!
'Exiting this function.'

提示

print 将显示不带引号的文本，但 return 将保留引号。

def falling(n, k):
    """Compute the falling factorial of n to depth k.

    >>> falling(6, 3)  # 6 * 5 * 4
    120
    >>> falling(4, 3)  # 4 * 3 * 2
    24
    >>> falling(4, 1)  # 4
    4
    >>> falling(4, 0)
    1
    """
    "*** YOUR CODE HERE ***"
    res = 1
    while k:
        res *= n 
        n -= 1
        k -= 1
    return res
