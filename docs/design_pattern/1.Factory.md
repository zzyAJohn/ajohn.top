::: tip UML类图举例
实现一个加减乘除的运算器。只需要输入运算符号，工厂就实例化出合适的对象，通过多态，返回父类的方式实现了计算器的结果。如果我们需要增加各种复杂运算，比如平方根，立方根，自然对数，正弦余弦等，只要增加相应的运算子类就可以了。
:::

@startuml

class Client {
  

}

abstract class Computer {
  +NumberA : double
  +NumberB : double
  +GetResult() : double
}

class Addition {
  
  +GetResult() : double
}

class Subtraction {
  
  +GetResult() : double
}

class Multiplication {
  
  +GetResult() : double
}

class Division {
  
  +GetResult() : double
}

Client -> Computer
Computer <|-- Addition
Computer <|-- Subtraction
Computer <|-- Multiplication
Computer <|-- Division


@enduml
