## 1. 规范脊椎姿势

::: note 处理人物姿势时，摄影师完全可以有很多不同的姿势选择，但是有一点确凿无疑，那就是拍摄对象的脊背必须保持端正。脊背端正是任何拍摄姿势取得理想效果的基础。置脊背姿势于不顾，照片的整体效果将会受到破坏。通过本章的学习，我非常希望读者了解这一点:在拍摄中，拍摄对象脊背姿势须端正得体，并尽可能显得修长。在很多情况下，常常需要使脊背呈现与平常不同的状态。
:::

### 1.1 颈椎

::: note 颈椎始于头骨的底部。
:::

::: note 以颈椎的底部为支点，将头部稍微向下倾斜，是拍摄的关键所在。这样一来，能够使得观众的视线顺着模特头部的倾斜方向转移到希望观众注意的地方。在处理颈椎姿势时，我通常遵循的方法是:拍摄个人肖像时，使拍摄对象颈椎挺直、头部后仰，使锁骨露出:当拍摄两个人，例如拍摄情侣时，可以使其中一方的颈椎向另一方倾斜，从而创造一种浪漫、富有情趣的图像效果。需要留意的是，身体的颈椎后仰或者倾斜时，必须以颈脖底部为枢轴点。假设颈脖和肩部的交接处是一圆点，头部则是绕着这个圆点旋转的。而除枢轴点之外，颈脖应尽可能地保持修长。头部上仰或低垂时，只有下巴需要挪动。
:::

### 1.2 胸椎

::: note 胸椎通常指颈部以下、腰部以上的脊椎部位。
:::

::: note 校正姿势时，我往往将人体的胸椎部分想象成一根长棍。它应尽可能地修长、笔直，而不曲、收缩。气质迷人的人，在走路时常常腰杆笔挺、身姿端正。但是拍照时，这往往还不够。得体的姿势要求人们脊椎挺直的同时，尽可能地延展、伸长。以下是我延展胸椎线条时常用的3个窍门。
- 让拍摄对象将头的顶部向上提，使其尽可能接近天花板，或者接近天空。在这里，我之所以强调上扬头顶的后部而不是前额，是因为如若强调后者，拍摄对象往往容易翘起下巴。而我们期望的是，挺直头部的同时。并不影响下巴的摆放。因此，这一技巧能够帮助模特很好地延长颈椎线条。
- 尽可能地吸气收腹，这样能让人感觉到浑身通透、舒展。然而，呼气容易产生相反的效果。我常常让拍摄对象深“吸”气，当他们“吸”到一半时，我则按下快门按钮，从而拍摄出自信满满、精神充沛的人像。当然，吸气程度必须在拍摄对象的吸气“舒适区”以内，如果让拍摄对象费尽解数地大口吸气，会让人看起来费力、不自然，反倒破坏了照片效果。所以，不要等到拍摄对象吸气困难的时候才开始按快门按钮。
- 让拍摄对象将肩膀向下、向后稍稍下垂几厘米。将肩膀向后挪动5厘米左右，对于延展颈部的线条有很好的效果，这不仅能突显人像头部、面部特点，还能够使拍摄对象看起来更加自信、有力。
:::

::: note 正确的姿势
站直身体，试着想象有一股力量将自己的头顶向上提，同时，将双肩降低、后靠，最后深吸一口气。这样一定会让自己感觉很不错，不是吗?这就是拍照时规范的姿势。拍摄时，按照这一规范作，视觉效果将有很大的改善。
:::

### 1.3 腰椎

::: note 腰椎位于人体臀部的上方。
:::

::: note 从摄影角度来说，由于腰椎具有明显的弧度，它是脊椎中最吸引人的部分。在处理人物姿势时，腰部的姿势是特别值得关注的，因为它连着人体的臀骨。而臀部很大程度上决定了一个女人是否优雅、性感。因此，我将腰部誉为“性感之源”。如果想使一个女人看起来迷人、性感，则必须格外注意其腰部姿势。
:::


::: note 人物腰椎部分有明显的弯曲度，看似一个侧置的微笑。为了方便记忆，可以这么理解:如果这部分看似在开心地微笑，那么它就是迷人的、性感的:反之，如果完全没有微笑的弧线，它则不够好看。
:::

::: note 平时腰部的弯曲通常比较自然。但在拍照时，需要适当地夸大一些弯曲度，将拍摄对象最好的一面呈现出来。这就是为什么拍出规范的姿势并不容易。如果拍摄对象并没有感到腰酸、费力，那么很可能是她的姿势没有摆对。
:::


::: tip 记忆卡片
要记住脊椎的每个部分是如何影响人物摆姿的，如下所示。
- 颈椎:倾斜颈椎有助于将头部指向期望观众注意的地方。颈椎应该尽可能地拉伸，以使头部挺直。
- 胸椎:胸椎是塑造整体姿势的关键。胸椎应尽可能地伸展，以使人物显得更加自信、有力。拍摄对象应该深吸气，才能使人像显得精神充沛。
- 腰椎:腰椎的处理主要用于修饰人物身线。它必须始终具有一定的弧度，以打造人物性感、迷人的特质。
:::

## 2. 重心分配及其影响

### 2.1 双腿交错转移身体重心

::: note 通过转移拍摄对象的身体重心，可以突显更多的人体曲线，并加强原有曲线的视觉效果。此外，通过转移重心。往往能使拍摄对象显得更加轻松、自然。当人物重心平均分配在双脚上时，人往往看起来呆板或是显得天真、孩子气。
:::

::: note 孩子们往往喜欢端端正正地用双脚站立，因为这样的姿势，能使重心平衡，并且让他们感到舒服、自在。所以,当拍摄小孩时，可以让他们将重心平均分配在双脚上。但是，拍摄成年人时，则不应采用这样的手法。成年人往往很关注自己的身形，所以，在拍摄中，他们往往希望自己看起来尽可能的苗条、修长。通过将拍摄对象的身体重心转移到其中一只脚上，尤其是到后脚上，往往可以达到这样的效果。
:::

::: note 使用这一拍摄手法时，其中一个很好的参照点是，想象人物双脚分别指向一条笔直的线条。如果这两条假想的直线最终是相交的，那么就正确地分配了两脚的重心。在这个专题讲解中，我只展示一些仅仅关注身体重心转移的拍摄手法，除此之外，其他部分保持不变。
:::

### 2.2 转移身体重心 双脚各指一方

::: note “转移身体重心，双腿并不交错”这一姿势非常适用于拍摄男性，或者使女性拍摄对象看起来强劲、有力。这是在拍摄中常见的、具有时尚性的手法。当拍摄青年学生或者时尚大片时，这一姿势都是非常不错的选择。采用这一拍摄技巧时，人物的身体重心应转移到后脚上。双脚站立时，分别指向一条笔直的线，且这两条不能为平行线。同时，最重要的一点在于，人物应将前脚的膝盖向着脚尖所指的方向略微弯曲。如若忽略这一点，整个姿势则会显得造作而不自然:如果将膝盖稍微向前弯曲，人物摆姿将呈现更加自然、舒适的效果，不再显得呆板、消极为了便于观察脚尖的指向，可以参考圈2.7中标注的红色线条。图中所示两条直线指向不同的方向，且按照它们的延长方向，它们是永不相交的。同时，尽管照片中并没有拍摄出人物膝盖，但是通过照片，能够明显感觉出人物的膝盖弯曲方向是和右腿的脚尖所指方向相一致的。
:::

特性：强劲、自信、坚定，适合时尚肖像

::: tip 记忆卡片
当拍摄对象将脚置于某物上，并弯曲膝盖时，大腿与地面形成的角度不宜超过45°。例如，大腿与地面形成的角度刚好是45°左右。为使姿势和谐、美观，该角度的最大限值是45”。如果大腿与地面形成的角度超过45°，拍摄姿势将会显得极其别扭，不自然。如图2.9所示，人物弯曲膝盖时，与地面形成的角度是不超过45°的。
:::


::: note 左起第二位男子是所有人当中姿势最为得体的。他所有的肢体动作都是规范的：首先，他的双脚脚尖方向并不是相对的，所以双脚脚尖方向上的两条直线是永不相交的；其次，他的身体重心主要放在后脚上，使其左脚处于放松的状态；第三，他较为放松的那条腿的膝盖，是朝着脚尖所指方向略微弯曲的。所有这些配合在一起，使其摆姿美观、大方，赏心悦目。
:::

::: note 将身体重心转移到**后脚**上，有助于使人物腰腹及以下部分远离镜头一些，从而突出人物的胸部而非肚子。
:::

### 2.3 坐姿中的重心分布

::: note 在拍摄对象采用坐姿时，有关重心分布的拍摄技巧也同样适用。但主要的区别在于:当拍摄对象站立时，人体的重心是在脚上的:当拍摄对象坐着时，身体重心是在臀线上的。将人体重心置于臀部的一大好处在于弱化了腹部给观众的视觉冲击，使观众的注意力转移到人体的腿部和躯干。因为如果拍摄对象的坐姿是正确的、优美的，其腿部和躯干部分往往比腰腹和臀部更加靠近镜头。这一特点在图2.13中得到了突出的体现。可以注意到，尽管照片中的人物是坐着的，她的背部是端正、挺査的，且其腰部呈现优美的弧线。同时，她交错着双腿，并隐约露出右腿的-小部分，展现出迷人、性感的女性气质。
:::

::: note 在图2.14所示照片中，我采用了同样的拍摄手法，但是，对其予以更加夸张的方式。既然将人体重心置于后腿上能够使人物显得更加放松、自然，何不将这一技巧用于拍摄人体坐姿?我让Britany 坐在台阶的边缘，将其身体重心置于左臀。如此一来，如图2.13所示，人物的大腿部分清晰可见，腿部线条也得到了加强，因此也更加凸显出人物的女性气质。当然，在拍摄男人时，采用这一姿势突出人体腿部则显得很不适合。
:::


## 3. 关节和直角

::: note 从本质上说，90°的直角往往能创造巨大的张力(视觉冲击力和艺术表现力)。比如，在古希腊、罗马的建筑中，所有的柱子都是与地面呈直角伫立的。因为，直角能够承受最大的重量，也是建筑艺术中，迄今为止最强大的角度架构。本章将把直角用于处理人体的拍摄姿势，并对其进行分析。
:::

### 3.1 弯曲关节 避免僵直

::: note 关节僵直
如若拍摄对象的关节部分星完全笔直的状态，往往给人以不自然的感觉。来做个小试验:当笔直地站立，手臂自然下垂时。我们会发现，此时的肘关节肯定略微弯曲的。接下来，对手臂施力，使其呈完全笔直的状态。如此一来，我们会感到这样需要花费很大的力气。如果人体本能地排斥某种特定的状态，则需要花很大的力气才能迫使它做到这一点。即便做到了，姿势往往也会显得非常造作、不自然。所以，请记住;拍摄中，干万别挑战人体的本能和极限，否则，人像将会显得焦虑，不自然。当人感到紧张不安时，他们的膝盖往往会显得十分僵直、别扭。所以，千万要避免这一现象。当然，某些时候，使得人体关节呈现笔直的状态，也不伤大雅。比如，当观众欣赏某些流行、时尚大片时，人体稍显不自然的姿势会引发观众潜意识里的不适感，使他们停下来仔细观看照片。因此，这一情况下，使拍摄对象关节僵直，往往会让观众进一步注意到人体的穿着、打扮等时尚细节。
:::


### 3.2 巧用手臂中的直角
::: note 通常情况下，摆姿势时，人物的躯干应该是面向观众的，且不应受到其他部分产生的视觉干扰，这样才能造就一张有魅力、受欢迎的照片。如果其他部分遮挡了人物躯干并产生视觉干扰，将有碍观众欣赏人像。同时，任何在人物躯干区域产生干扰的因素，都将影响观众对于人物脸部的关注。由于人物手臂和躯干部分同样重要，而直角往往更容易吸引人们的注意力，因此摄影师在处理人物手臂时应该格外关注并巧妙运用这一点。学会将拍摄对象的手臂合理地成直角弯曲是一项必须掌握的拍摄手法。它无所谓好坏，但是通过这一手法，可以创造出某种特定的照片效果，仅此而已。
:::

::: note 不过，无论什么时候，如若拍摄对象的手臂是成直角弯曲的，它必须是置于某物之上，而不应该空着手，或是拿着某物增加手中的重量。否则，人像将显得累赘、不优雅。
:::

### 3.3 情侣照中巧用手臂直角

::: note 作为拍摄者，必须知道自己期望通过一定的人体姿势获得怎样的效果。如果期望拍摄出强劲、有力的姿势，而并没有特别看重流畅、自然，那么尽管让人物将手臂成90°弯曲。但是需要注意的是，如果这样做，则需加强对拍摄对象脸部的处理，使其避免来自手臂的视觉干扰。如若不然，观众的目光往往会游离到除脸部之外的其他部位。
:::

### 3.4 采用直角塑造强劲或灵巧的姿势

::: tip 记忆卡片
如果将拍摄对象的手臂成直角弯曲，应该使其手臂搁置在某物之上。如果他/她手里握着某物却没有任何支撑物，这将使人物姿势显得尴尬、笨拙而不自然。
90”角往往传达出强劲有力的感觉，且引人注目。如果希望某个姿势看起来干练、有力，同时希望减弱观众对于人物脸部的注意，可以采用90”角修饰人物姿势。但是，如果期望创造更加流畅、优雅的姿势，同时加强观众对于人物脸部的关注，则应尽量减少采用90”角。
锐角(即小于90”的角)往往是惹眼、视觉冲击力强的角十分适合拍摄时尚写真，或者使拍摄姿势更加动感。
钝角(即大于90”的角)往往呈现平静、优雅的视觉效果，因此往往用于塑造浪漫、舒畅的人体姿势。
:::

### 3.5 巧用手腕手指中的直角
